Приложение "test your knowledge" предназначено для помощи начинающим фронтенд разработчикам в подготовке к собеседованиям.

Нажимая кнопку "Получить вопрос" Вам будет задан рандомный вопрос из наиболее часто задаваемых вопросов на собеседованиях. 

После Вы можете проверить себя, нажав кнопку "Получить ответ".

Таким образом, Вы сможете сами тренировать себя. Всем Удачи! =)
